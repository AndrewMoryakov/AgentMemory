# Support

## What To Use For What

- bug reports: use GitHub issues
- feature ideas: use GitHub issues
- security reports: follow [SECURITY.md](SECURITY.md)
- architecture/context: start with [docs/START_HERE.md](docs/START_HERE.md)

## Before Opening An Issue

Please try:

- `agentmemory doctor`
- `agentmemory doctor-clients --compact`
- `agentmemory mcp-smoke`
- confirming which provider is active
- confirming whether the problem is local runtime, provider behavior, or client integration

## What Helps A Lot

Include:

- operating system
- Python version
- active provider
- exact commands
- whether the issue is in CLI, HTTP API, MCP, or client integration
- relevant error output

## Project Status

AgentMemory is a `public alpha`.

Clear bug reports, docs fixes, examples, and tests are especially useful right now.
